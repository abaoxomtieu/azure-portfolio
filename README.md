# Azure Architecture Portfolio

A brutalist, motion-driven portfolio site showcasing Azure Certification Guides and Practice Architectures.

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS + Brutalist Design System
- **Animation**: Framer Motion
- **Content**: Markdown + Mermaid.js Diagrams

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000)

3. **Build for Production**:
   ```bash
   npm run build
   npm run start
   ```

## Project Structure
- `content/`: Markdown files for Certs and Practice.
- `src/app/`: Next.js App Router pages.
- `src/components/`: UI Components (Hero, ProjectCard, etc.).
- `src/lib/`: Utilities (Markdown parser).
