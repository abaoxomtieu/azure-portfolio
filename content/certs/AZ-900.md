# AZ-900: Microsoft Azure Fundamentals - Study Guide

Tài liệu tổng hợp các kiến thức nền tảng về Cloud và Azure cho kỳ thi AZ-900.

---

## 1. Cloud Concepts (Khái niệm Điện toán đám mây)

### Cloud Models (Mô hình đám mây)
- **Public Cloud**: Hạ tầng dùng chung, Microsoft quản lý phần cứng. Rẻ nhất (VD: Azure, AWS).
- **Private Cloud**: Đám mây riêng, dedicated cho 1 tổ chức. Tự quản lý hoặc thuê. Bảo mật cao nhất nhưng đắt.
- **Hybrid Cloud**: Kết hợp cả Public và Private. Linh hoạt nhất.

### Service Models (Mô hình dịch vụ)
- **IaaS (Infrastructure as a Service)**: Thuê phần cứng (VM, Network, Disk). Bạn quản lý OS và App. (VD: Azure VM). >> *Linh hoạt nhất*.
- **PaaS (Platform as a Service)**: Thuê nền tảng (Runtime, DB). Bạn chỉ lo Code và Data. (VD: App Service, SQL Database). >> *Tập trung phát triển*.
- **SaaS (Software as a Service)**: Dùng phần mềm như dịch vụ. Không quản lý gì cả. (VD: Microsoft 365, Gmail). >> *Tiện lợi nhất*.

### Expenditure Models (Mô hình chi phí)
- **CapEx (Capital Expenditure)**: Chi phí vốn. Mua server 1 lần, khấu hao dần. (Mô hình On-premise).
- **OpEx (Operational Expenditure)**: Chi phí vận hành. Thuê bao nhiêu trả bấy nhiêu (Pay-as-you-go). (Mô hình Cloud).

## 2. Core Azure Services (Các dịch vụ cốt lõi)

### Compute (Tính toán)
- **Azure Virtual Machines (VM)**: Máy ảo IaaS.
- **Azure App Service**: PaaS để host web/api.
- **Azure Container Instances (ACI)**: Chạy container đơn lẻ, nhanh (Burst).
- **Azure Kubernetes Service (AKS)**: Quản lý cụm container (Orchestration).
- **Azure Functions**: Serverless, chạy code theo sự kiện.

### Networking (Mạng)
- **Virtual Network (VNet)**: Mạng riêng ảo.
- **VPN Gateway**: Kết nối mã hóa qua internet.
- **ExpressRoute**: Kết nối riêng (Private line) không qua internet.

### Storage (Lưu trữ)
- **Blob Storage**: Lưu file, ảnh, video (Object storage).
- **Disk Storage**: Ổ cứng cho VM.
- **File Storage**: File share (SMB) dùng chung.

### Database
- **Cosmos DB**: NoSQL toàn cầu.
- **Azure SQL Database**: SQL Server bản nhen nhóm (PaaS).
- **Azure Database for MySQL/PostgreSQL**: Dịch vụ database mã nguồn mở được quản lý.

### IoT (Internet of Things)
- **Azure IoT Hub**: Trung tâm điều khiển, kết nối hàng tỷ thiết bị IoT.
- **Azure IoT Central**: Giải pháp SaaS trọn gói để quản lý thiết bị IoT mà không cần code nhiều.
- **Azure Sphere**: Giải pháp bảo mật phần cứng cho vi điều khiển (MCU) của thiết bị IoT.

### Big Data & Analytics
- **Azure Synapse Analytics**: Kho dữ liệu (Data Warehouse) không giới hạn, kết hợp Big Data và Data Warehousing.
- **Azure HDInsight**: Chạy các framework nguồn mở như Hadoop, Spark, Kafka trên Azure.
- **Azure Databricks**: Nền tảng Apache Spark tối ưu hóa cho Azure, chuyên dùng cho Data Engineering và Data Science.

## 3. Azure Architecture (Kiến trúc Azure)

### Physical Infrastructure
- **Regions (Vùng)**: Tập hợp các data center (VD: East US, Southeast Asia).
- **Availability Zones (AZ)**: Các DC độc lập về điện/làm mát trong cùng 1 Region. (Chống sập DC).
- **Region Pairs**: Cặp vùng luôn đi cùng nhau để replicate dữ liệu (VD: East US luôn pair với West US).

### Management Infrastructure
- **Resources**: Tài nguyên cụ thể (1 con VM, 1 cái DB).
- **Resource Groups (RG)**: Thùng chứa mang tính logic để gom nhóm resource cùng vòng đời (Lifecycle).
- **Subscriptions**: Đơn vị thanh toán và quản lý quota.
- **Management Groups**: Gom nhóm các Subscription để áp dụng Policy chung.

### Sovereign Regions (Các vùng đặc thù)
- **Azure China**: Dành riêng cho thị trường Trung Quốc (do 21Vianet vận hành).
- **Azure Government (US)**: Dành cho chính phủ Mỹ.

## 4. Identity, Governance, Privacy & Compliance

### Identity (Định danh)
- **Microsoft Entra ID**: Dịch vụ quản lý danh tính (Identity Provider).
- **SSO (Single Sign-On)**: Đăng nhập 1 lần vào được mọi app.
- **SSO (Single Sign-On)**: Đăng nhập 1 lần vào được mọi app.
- **MFA (Multi-Factor Authentication)**: Bảo mật nhiều lớp (Something you know, Something you have, Something you are).
- **Conditional Access**: "Nếu... thì..." (Nếu user đăng nhập từ IP lạ, thì bắt nhập OTP).

### Security (Bảo mật)
- **Zero Trust**: Nguyên tắc "Không tin bất kỳ ai", luôn xác thực lại.
- **Defense in Depth**: Bảo mật nhiều lớp (Vật lý > Danh tính > Chu vi > Mạng > Tính toán > Ứng dụng > Dữ liệu).
- **Azure Key Vault**: Lưu trữ password, key, certificate an toàn.
- **Azure Sentinel**: Hệ thống SIEM (Quản lý sự kiện thông tin bảo mật) dùng AI để phát hiện tấn công.
- **Azure Dedicated Host**: Thuê nguyên server vật lý (không share với ai) để đảm bảo compliance.


### Governance (Quản trị)
- **Role-Based Access Control (RBAC)**: Ai được làm gì? (Who can do what?).
- **Resource Locks**: Chống xóa nhầm.
- **Azure Policy**: Ép buộc tuân thủ quy tắc (VD: Không được tạo VM ở vùng US).
- **Azure Blueprints**: Bản thiết kế mẫu (gồm Policy, RBAC, ARM Template) để tạo môi trường mới nhanh và chuẩn.

### Cost Management (Quản lý chi phí)
- **Pricing Calculator**: Ước tính giá **trước khi** mua.
- **TCO Calculator**: So sánh giá Cloud vs On-premise để thấy tiết kiệm bao nhiêu.
- **Microsoft Cost Management**: Theo dõi chi tiêu thực tế và set **Budget** (ngân sách) để cảnh báo khi tiêu lố.
