## 1. Plan and Manage an Azure AI Solution (20–25%)

### Chọn đúng service (Microsoft Foundry Services)

- **Generative AI** → Azure OpenAI / Microsoft Foundry (model catalog, prompt flow, RAG).
- **Computer vision** → Azure AI Vision, Custom Vision, Video Indexer, Vision Spatial Analysis.
- **NLP** → Azure AI Language (CLU, QA, Translator, text analytics).
- **Speech** → Azure AI Speech (STT, TTS, translation, custom speech, intent/keyword).
- **Information extraction** → Document Intelligence, Content Understanding.
- **Knowledge mining** → Azure AI Search (index, skillset, semantic, vector).

### Tạo và triển khai Azure AI resource

- **Multi-service resource** vs **single-service resource**: Multi-service dùng chung cho Vision, Language, Speech (tiện; quota chung). Single-service tách từng loại (kiểm soát chi phí/quota riêng).
- **Responsible AI**: Thiết kế solution tuân thủ Fairness, Reliability & Safety, Privacy & Security, Inclusiveness, Transparency, Accountability.
- **Deployment**: Chọn model phù hợp (GPT-4 vs 3.5, vision model…), deploy qua portal/SDK; container deployment cho edge/on-prem.
- **SDK & API**: Cài SDK (Python/C#), gọi REST API; xác định default endpoint và key/Managed Identity.
- **CI/CD**: Tích hợp Azure AI vào pipeline (deploy model, update config).

### Quản lý, giám sát, bảo mật

- **Monitor**: Diagnostic Settings đẩy log sang Log Analytics; theo dõi latency, errors, throttling.
- **Cost**: Theo dõi usage, quota; tách resource theo môi trường nếu cần.
- **Keys**: Không hardcode key; dùng **Managed Identity** hoặc Key Vault.
- **Authentication**: Managed Identity, API key (rotation), VNet/private endpoint để hạn chế truy cập từ internet.

### Implement AI responsibly

- **Content moderation**: Dùng Content Safety (hoặc tương đương) cho nội dung user-generated.
- **Responsible AI insights**: Bật content safety, filters, blocklists.
- **Prompt shields & harm detection**: Bảo vệ prompt và phát hiện nội dung gây hại.
- **Governance**: Framework rõ ràng (policy, review, monitoring).

**Lưu ý:** Đề có thể hỏi “chọn service nào cho use case X” – cần nắm rõ Vision vs Language vs Speech vs Document Intelligence vs Search vs OpenAI.

## 2. Implement Generative AI Solutions (15–20%)

### Microsoft Foundry

- Plan và chuẩn bị (data, use case, model).
- Deploy hub, project, resources trong Foundry.
- Deploy model generative phù hợp (chat, code, image).
- **Prompt Flow**: Thiết kế flow (prompt → model → evaluation).
- **RAG**: Grounding model với data của bạn (Azure AI Search thường dùng làm retriever).
- Evaluate model và flow; tích hợp vào app qua Foundry SDK; prompt templates.

### Azure OpenAI (trong Foundry Models)

- Provision Azure OpenAI resource (trong Foundry).
- Chọn và deploy model (GPT-4, GPT-3.5, DALL-E, embedding, Whisper).
- Submit prompt (chat, completion); generate code/natural language.
- **DALL-E**: Tạo ảnh từ mô tả (text-to-image).
- Tích hợp Azure OpenAI vào app (SDK, REST).
- **Large multimodal models**: Ảnh + text input.

### Tối ưu và vận hành

- **Parameters**: Temperature, Top P, max_tokens để điều khiển hành vi generative.
- **Monitoring**: Diagnostic settings, performance, resource consumption.
- **Scale**: Tuning capacity, deployment options; cập nhật foundational model khi cần.
- **Tracing & feedback**: Thu thập feedback, theo dõi chất lượng.
- **Model reflection**: Kỹ thuật cải thiện output (self-check, refine).
- **Container**: Deploy container cho local/edge.
- **Orchestration**: Kết hợp nhiều model (routing, chain).
- **Prompt engineering**: System message, few-shot, zero-shot.
- **Fine-tuning**: Fine-tune model trên data riêng.

**Lưu ý:** RAG = Retrieval (vd: Azure AI Search) + Augmented prompt + Generation; cần nắm luồng: query → search → prompt + context → LLM.

## 3. Implement an Agentic Solution (5–10%)

- **Agent**: Thành phần tự động dùng LLM + tools (search, code, API) để hoàn thành task.
- **Use cases**: Chatbot có hành động, automation, multi-step reasoning.
- Cấu hình resource (Foundry, OpenAI, Search…).
- **Microsoft Foundry Agent Service**: Tạo agent (GUI/config).
- **Microsoft Agent Framework**: Agent phức tạp, multi-agent, orchestration, multi-user, autonomous.
- Test, tối ưu, deploy agent.

**Lưu ý:** Phần này ít % nhưng cần biết agent là gì và Foundry Agent Service / Agent Framework dùng để làm gì.

## 4. Implement Computer Vision Solutions (10–15%)

### Phân tích ảnh (Azure Vision – Foundry Tools)

- Chọn visual features (tags, description, OCR, handwritten).
- Detect objects, generate image tags.
- Include image analysis features trong request; interpret response.
- **OCR**: Extract text từ ảnh (Azure Vision).
- **Handwritten**: Convert chữ viết tay thành text.

### Custom Vision

- **Image Classification** vs **Object Detection**: Classification = gán nhãn cho cả ảnh; Detection = vị trí + nhãn từng object.
- Label images, train (classification hoặc detection), đánh giá metrics, publish, consume (API/SDK).
- Build custom vision model code-first (SDK).

### Video

- **Azure AI Video Indexer**: Extract insights từ video hoặc live stream (faces, OCR, sentiment, scenes).
- **Azure AI Vision Spatial Analysis**: Detect presence và movement của người trong video.

**Lưu ý:** Custom Vision = train riêng; Azure AI Vision (Image Analysis) = prebuilt. Document Intelligence = document/form; Vision = ảnh tổng quát.

## 5. Implement Natural Language Processing Solutions (15–20%)

### Phân tích và dịch văn bản

- Key phrase extraction, entity recognition, sentiment, language detection, PII detection.
- **Azure Translator** (Foundry Tools): Dịch văn bản, tài liệu.

### Speech (Azure Speech – Foundry Tools)

- Generative AI speaking (vd: conversational TTS).
- **Text-to-Speech (TTS)**, **Speech-to-Text (STT)**.
- **SSML**: Cải thiện TTS (phát âm, tốc độ, giọng).
- **Custom speech**: Train model nhận diện giọng/âm domain riêng.
- **Intent & keyword recognition**: Speech → intent/keyword.
- **Speech translation**: Speech-to-speech, speech-to-text translation.

### Custom language models

- **CLU (Conversational Language Understanding)** – thay LUIS:
  - Intent, entity, utterances; train, evaluate, deploy, test; optimize, backup, recover.
  - Consume từ client app.
- **Question Answering** – thay QnA Maker:
  - Tạo project QA; thêm Q&A pairs, import sources; train, test, publish knowledge base.
  - Multi-turn conversation; alternate phrasing, chit-chat; export; multi-language.
- **Custom translation**: Train, improve, publish custom translation model.

**Lưu ý:** CLU = intent/entity; QA = Q&A từ tài liệu. Translator = dịch; Language = sentiment, entity, key phrase, PII.

## 6. Implement Knowledge Mining and Information Extraction (15–20%)

### Azure AI Search

- Provision resource; tạo **index**, **data source**, **indexer**.
- **Skillset**: Enrichment (OCR, entity, translation, custom skill).
- **Custom skill**: Azure Function (Python/C#) nhúng vào pipeline.
- Create và chạy indexer.
- **Query**: Syntax, sort, filter, wildcard.
- **Knowledge Store**: Projections (file, object, table).
- **Semantic search**, **vector store** (vector search).

### Azure AI Document Intelligence (Foundry Tools)

- Provision Document Intelligence resource.
- **Prebuilt models**: Invoice, Receipt, ID, Layout, Read, v.v.
- **Custom model**: Train, test, publish custom document model.
- **Composed model**: Gộp nhiều model (prebuilt + custom) vào một composed model.

### Azure Content Understanding (Foundry Tools)

- **OCR pipeline**: Extract text từ ảnh/tài liệu.
- Summarize, classify, detect attributes.
- Extract entities, tables, images.
- Process documents, images, videos, audio.

**Lưu ý:** Search = indexing pipeline (crack → skillset → index) + query. Document Intelligence = form/document extraction. Content Understanding = broader doc/video/audio understanding.

## Lưu ý chung khi ôn AI-102

1. **Tên dịch vụ**: Exam đã dùng “Microsoft Foundry” và “Foundry Tools” (Azure AI Vision in Foundry Tools, Azure Speech in Foundry Tools…). Vẫn cần biết tên cũ (Cognitive Services, LUIS, QnA Maker) vì tài liệu cũ có thể còn dùng.

4. **RAG**: Retriever (thường Azure AI Search) + prompt (context) + LLM; biết cách ghép nối.
